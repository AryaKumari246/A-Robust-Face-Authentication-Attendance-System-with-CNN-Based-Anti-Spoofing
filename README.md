# A-Robust-Face-Authentication-Framework-with-CNN-Based-Anti-Spoofing-for-Intelligent-Attendance-Systems

To develop a secure and automated attendance system that prevents proxy attendance using face recognition and anti-spoofing techniques using deep learning,  
facial recognition, and liveness detection. 
It reduces manual work, improves transparency, and is suitable for educational institutions and organizations ..
It verifies student liveness through eye-blink detection and anti-spoofing techniques, achieving 95.2% recognition accuracy and 99.6% spoof detection accuracy ..
